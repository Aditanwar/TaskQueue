Raw Redis Queuing Mechanics

A lightweight alternative bypassing Celery using atomic LPUSH & BLPOP instructions.

How it Works

Unlike Celery, which packs a heavy abstraction layer, a raw queue relies entirely on Redis native lists. Producers append to lists using LPUSH, and workers subscribe and sleep on socket connections using blocking BLPOP operations.

[FastAPI Producer] --(LPUSH)--> [ Redis List: raw_queue:jobs ] <--(BLPOP)-- [Raw Python Worker]

Test it locally

To test the lightweight Raw Redis comparison queues, open two command prompt windows in your local workspace and execute:

Terminal A: Launch the Consumer Worker
python raw_redis/worker.py
Terminal B: Submit Jobs via Producer Client
python raw_redis/client.py
